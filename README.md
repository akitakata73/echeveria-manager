# エケベリア管理 PWA v3

## v3の主な変更点

- 置き場所の「種類」を廃止し、「方角」に変更
  - 東向き / 西向き / 南向き / 北向き
- 設置場所をプルダウン化
  - 室内 / 屋外
- 乾きやすさを5段階化
  - 1：かなり乾きやすい
  - 2：やや乾きやすい
  - 3：普通
  - 4：やや乾きにくい
  - 5：かなり乾きにくい
- 風に「不定期」を追加
- v1/v2のlocalStorageデータを自動移行

## 端末間同期について

このPWAは端末ごとのSafari内に保存されます。
黒iPhoneと白iPhoneのデータは自動同期されません。

別端末へ移す場合は、設定 → 出力 → JSONバックアップをコピーし、
別端末側で設定 → 復元からJSONを貼り付けて復元してください。
