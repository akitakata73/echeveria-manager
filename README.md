# エケベリア管理 PWA v25 Recovery

## 修正した原因

- v24復旧版では、内部の STORAGE_KEY が古い値のまま残っていた
- 設定画面で APP_VERSION を表示していたが、APP_VERSION の定義が入っていなかった
- そのため、設定タブを開くと JavaScript エラーで停止していた

## v25で追加したこと

- APP_VERSION を明示定義
- STORAGE_KEY を v25_recovery 専用に修正
- v1〜v24までの保存キーを全探索
- 株数・履歴数・写真数・マスター数から、最も内容が多いデータを自動選択
- 設定画面に以下を表示
  - 現在のバージョン
  - 読込元キー
  - 復旧スコア
  - 保存データ診断一覧

## 確認手順

1. GitHubへ上書き
2. `?v=25-recovery` で開く
3. 設定画面を開く
4. 「読込元」と「保存データ診断」を確認
5. 株・履歴・写真が戻っていれば、すぐJSONバックアップを取る
