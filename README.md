# エケベリア管理 PWA v33 Save Visibility

## v33の目的

保存方式はv32のまま維持し、保存状態の見える化を強化した版です。

## 主な変更点

- 保存サイズを表示
- 写真枚数を表示
- 最終保存成功時刻を明確化
- 保存先を「固定キー＋直前データ」として表示
- 「previousから復元」を「直前データに戻す」に表示変更
- 保存サイズ警告は追加しない
- 旧復旧候補欄は復活させない

## 保存方式

- 固定キー：echeveria_pwa_data
- 直前データ：echeveria_pwa_previous_data

## 注意

SafariのWebサイトデータ削除を行うと、固定キー・直前データ・緊急バックアップはまとめて消える可能性があります。
アプリ更新前や写真追加後は、必要に応じてJSONバックアップも併用してください。
