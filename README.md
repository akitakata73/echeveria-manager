# エケベリア管理 PWA v24 Recovery

## v24の主な変更点

- 過去バージョンの保存データ探索を強化
- v1〜v23までの localStorage 保存キーを全探索
- 株数・履歴数・写真数・マスター数から、最も内容が多いデータを優先して読み込み
- 読み込んだデータを v24 保存キーへ移行
- 設定画面に現在バージョンと読込元キーを表示
- v23で履歴が消えたように見える症状への復旧対応版

## 重要

これはロールバックではなく、過去データを探して現在版へ移行する復旧版です。
端末内に過去のlocalStorageデータが残っていれば復旧できる可能性があります。

## 確認手順

1. v24をGitHubへアップロード
2. iPhoneで `?v=24-recovery` を付けて開く
3. 設定画面で「現在のバージョン：v24」と「読込元」を確認
4. 株・履歴・写真が戻っているか確認
5. 戻っていればJSONバックアップをすぐ取得
