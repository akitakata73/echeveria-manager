# エケベリア管理 PWA v18

## v18の主な変更点

- v17の半自動バックアップ管理機能を維持
- 黄色背景＋黒ジョウロの益田カラー版アイコンを正式同梱
- 今後の標準アイコンを以下に固定
  - icon-192.png
  - icon-512.png

## v17から維持している主な機能

- ホーム画面と設定画面にバックアップ状況を表示
- 前回バックアップ日時を記録
- JSONバックアップをコピーしたタイミングで「バックアップ済み」として記録
- データ更新後は「バックアップ推奨」状態に変更
- 7日以上バックアップがない場合に注意表示
- 写真枚数もバックアップ状況に表示

## 運用方針

- 通常時：週1回バックアップ
- 写真追加・新規株登録・大きな編集後：当日バックアップ
- アプリ更新前：必ずバックアップ

## 注意

Google Driveへの完全自動保存ではありません。
JSONをコピー後、Google Drive・メモ・ファイル等へ手動保存する半自動型です。
