# エケベリア管理 PWA v5

## v5の主な変更点

- 画面上の絵文字アイコン表示を削除
  - 株一覧
  - 株詳細
  - 履歴・記録
- 写真がある株は最新写真を表示
- 写真がない株は「写真なし」と表示
- 株情報に「個体区分」を追加
  - 成株
  - 葉挿し
  - 頭株
  - 元株
  - 子株
  - 実生
  - その他
- 状態から「葉挿し」を外し、葉挿しは個体区分で扱う構成に整理
- v1〜v4のlocalStorageデータを自動移行
  - 既存データで状態が「葉挿し」だった株は、個体区分「葉挿し」・状態「良好」に移行
