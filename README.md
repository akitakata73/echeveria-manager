# エケベリア管理 PWA v44 Daily Usability UI

## v44の目的

保存方式・写真方式には触れず、毎日の確認で使う画面の視認性と操作性を改善した版です。

## 主な変更点

- ホームに「確認目安超過」ブロックを独立表示
- ホームに「近日確認」ブロックを追加
- ホームに「最近の記録」ブロックを追加
- 株カードの次回確認日を見やすく表示
- 株詳細の上部に「確認サマリー」を追加
  - 次回確認
  - 最終水やり
  - 状態
  - 置き場所
- 株詳細の操作ボタンを分離
  - 日常操作：水やり／観察／写真
  - 管理操作：株情報編集／クローン作成
- 保存系・写真系はv43までの仕様を維持

## 維持している固定ルール

- 保存キー：echeveria_pwa_data
- 直前データ：echeveria_pwa_previous_data
- 写真本体：IndexedDB
- 旧キー自動採用なし
- 保存データ診断・復旧候補欄なし
- localStorage.clear() 未使用
- v40 Safe Commit の保存確定処理を維持
- v41 Photo Stability の写真保存安定化を維持
