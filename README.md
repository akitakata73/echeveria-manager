# エケベリア管理 PWA v65 Leaf Group Base Fix

## 主な変更点
- 葉挿し群モードを追加
- 株一覧から「自宅葉挿し」「実家葉挿し」を作成できる導線を追加
- 葉挿し群は、品種ごとに「発根前（枚）」と「1号ポット（個）」を管理
- 水やり・観察は葉挿し群単位で従来画面を使用
- 自宅葉挿しは「自宅　棚」、実家葉挿しは「実家その他」に配置
- 葉挿し群は配置一覧にも出るが、マス指定なしの棚扱い
- 診断用データに葉挿し内訳を出力
- 既存の葉挿し個別株は自動統合しない
- クローン作成時は葉挿し群情報をコピーしない
- 既存の通常株処理に影響しないよう、leafGroupMode 分岐で追加
