# エケベリア管理 PWA v12

## v12の主な変更点

- 株詳細のボタン配置を整理
  - 1列目：水やり／観察
  - 2列目：株情報編集／写真
  - 写真追加は写真画面内に集約
- ホーム画面を日常運用向けに整理
  - 今日確認する株
  - 期限超過の株
  - 明日確認する株
  - 最近の記録
- カレンダー下部の全株一覧をグルーピング
  - 期限超過
  - 今日
  - 明日
  - 今週
  - 来週以降
  - 未設定
- カレンダーに簡易フィルターを追加
  - 要観察
  - 状態
  - 個体区分
- 表示上のCSV出力ボタンを削除
  - 内部関数は維持
  - JSONバックアップは継続
- 水やり画面・株編集画面から「水やり手法」入力を削除
- 計算理由を簡易表示＋詳細表示に整理

## 注意

今回のv12は機能追加よりも日常運用のUI整理を優先した版です。
