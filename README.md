# エケベリア管理 PWA v4

## v4の主な変更点

- 株情報に写真を追加できるように変更
- 新規株登録時に写真を登録可能
- 水やり記録時に写真を追加可能
- 株詳細から写真一覧をアルバム表示
- 最新写真を株一覧・株詳細のスクエアアイコンとして表示
- 写真は中央スクエアで切り抜き、512pxに圧縮して保存
- 新規株登録のアイコン入力を削除
- 水やりmL入力を削除
- 置き場タブを削除し、置き場所管理は設定タブに集約
- 風：風強め → 強風あり、弱め → 中くらい
- 乾きやすさ：数字と「：」を削除
- 光を5段階化

## 注意

写真はlocalStorage内に保存されます。写真が増えると容量を圧迫します。
定期的にJSONバックアップを保存してください。
