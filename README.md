# エケベリア管理 PWA v21

## v21の主な変更点

- v20で発生した画面が表示されない不具合を修正
- 原因：deleteMaster() と copyText() の旧処理が重複して残り、JavaScriptが停止していた
- 安定版v19をベースに、v20で予定していた事故防止・表示整合性改善を再適用

## 再適用した改善

- 写真選択時も入力保護の対象に追加
- 要観察を手動プルダウンから読み取り専用の自動表示に変更
- 水やり・観察・株情報編集を開いた時点で要観察表示を状態と同期
- JSONバックアップは、先にバックアップ日時を更新してからJSONを生成・コピー
- マスターの最後の1件は削除不可
- マスター削除判定はアーカイブ株も含めた使用数で判定
- カレンダーの日付件数をフィルター後の件数に変更
- 右上更新ボタンで、株詳細・写真・水やり・観察など現在表示中のページを更新
- 設定画面に現在バージョンを灰色文字＋下線で表示
