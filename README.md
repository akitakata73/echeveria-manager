# エケベリア管理 PWA v22

## v22の主な変更点

- 株詳細画面に「クローン作成」ボタンを追加
  - 配置：株情報編集／写真／クローン作成
- クローン作成時、元株の情報をコピーした新規株登録画面を表示
- クローン初期名は「元の管理名 クローン」
- クローン時にコピーするもの
  - 品種
  - 鉢
  - 用土
  - 置き場所
  - 位置メモ
  - 個体区分
  - 状態
  - メモ
- クローン時にコピーしないもの
  - 写真
  - 最終水やり日
  - 最終水やり量
  - 次回確認日
- クローン時のメモには、元メモの末尾に「〇〇からクローン作成」を追記
- 個体区分に「中株」を追加
  - 並び順：成株／葉挿し／頭株／元株／子株／中株／実生／その他
- 個体区分ごとの基準日数を見直し
  - 葉挿し：3日
  - 実生：3日
  - 子株：4日
  - 中株：5日
  - 頭株：5日
  - 元株：6日
  - 成株：7日
  - その他：7日

## v21相当の安全修正も維持

- 写真選択時の入力保護
- 要観察の読み取り専用自動表示
- バックアップコピー順序修正
- マスター削除安全化
- 設定画面のバージョン表示

## 注意

頭株は状態に関係なく5日基準です。
成株の基準7日は変更していません。
