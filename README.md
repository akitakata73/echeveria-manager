# エケベリア管理 PWA v7

## v7の主な変更点

- 記録と設定の間にカレンダータブを追加
- カレンダーの日付を押すと、その日の確認対象株を表示
- カレンダー下部に全株を次回確認日順で表示
- 株一覧の「お迎え日」表示を「管理日数」に変更
- 状態に「発根管理中」を追加
- 発根管理中の状態補正を -0.5日に設定
- 休眠気味の補正を +1.0日に修正
- 要観察を状態から分離し、ON/OFFフラグ化
- 既存データで状態が「要観察」の株は、状態「良好」＋要観察ONへ自動移行
- 要観察フィルターは、要観察ONまたは良好以外の株を表示

## 注意

今回もサーバ・同期機能はありません。
各端末ごとにデータは別管理です。
