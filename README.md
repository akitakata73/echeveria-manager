# エケベリア管理 PWA v20

## v20の主な変更点

- 写真選択時も入力保護の対象に追加
- 要観察を手動プルダウンから読み取り専用の自動表示に変更
  - 良好：自動OFF
  - 良好以外：自動ON
- 水やり画面・観察画面・株情報編集画面を開いた時点で要観察表示を状態と同期
- JSONバックアップのコピー時、先にバックアップ日時を更新してからJSONを生成・コピーするように修正
- マスターの最後の1件は削除不可に変更
- マスター削除判定はアーカイブ株も含めた使用数で判定
- カレンダーの日付件数をフィルター後の件数に変更
- 右上更新ボタンで、株詳細・写真・水やり・観察など現在表示中のページを更新
- 設定画面に現在バージョンを小さな灰色文字＋下線で表示

## 注意

入力保護の確認ダイアログはSafari標準表示です。
ボタン色・ボタン文言の完全な制御はしていません。
