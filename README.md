# エケベリア管理 PWA v73 Watering Placement Focus Fix

## 主な変更点
- かんたん水やり・通常水やりの保存成功後、対象株の配置位置へ自動遷移
- PMTトレー株
  - 対象マスまで自動スクロール
  - 一時的に緑枠で強調
  - 水やり当日は従来どおり 0日・水色表示
- 自宅　棚・実家その他
  - 対象株の行まで自動スクロール
  - 一時的に緑枠で強調
- 配置未指定、配置単位が不明、トレー位置が不完全な株
  - 無理に配置一覧へ遷移せず、株情報ページに戻る
- 水やり入力画面上の「配置一覧へ戻る」ボタンは削除
- 保存処理・履歴処理・次回確認日計算には触れない
