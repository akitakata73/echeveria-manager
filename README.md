# エケベリア管理 PWA v10

## v10の主な変更点

- 鉢・用土の乾きやすさ補正が入ったことに合わせ、短期管理系の基本日数を引き上げ
  - 葉挿し：2日 → 3日
  - 発根待ちの頭株：4日 → 5日
  - 胴切り後の元株：5日 → 6日
  - 成株：7日のまま変更なし
- 状態変更時の要観察フラグを自動化
  - 状態が「良好」：要観察OFF
  - 状態が「良好」以外：要観察ON
- 株情報編集、水やり、観察の各画面で状態変更に合わせて要観察表示を自動切替

## 注意

要観察は手動管理から、状態に連動する補助フラグへ寄せています。
「良好だが念のため要観察」という特殊運用は、今後必要であれば別途「重点確認」などの独立フラグで追加する方が安全です。
