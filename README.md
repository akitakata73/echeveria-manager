# エケベリア管理 PWA v39 Photo DB Safe Save

## v39の目的

1MB超過による保存失敗と、アプリ更新時のデータ欠損リスクを下げるため、写真本体を本体JSONから切り離し、保存処理を安全化した版です。

## 主な変更点

- 写真本体を localStorage の本体JSONから切り離し、IndexedDBへ保存
- localStorageには写真メタ情報のみ保存
- 既存のインライン写真データは、起動後に可能な限りIndexedDBへ移行
- 写真追加時に 768px 四方のJPEGへ軽量化
- 本体保存は一時キーへ保存→検証→previous退避→本番昇格の順に変更
- 保存失敗時は本番データを上書きしない
- アプリ本体更新ボタンは、保存・緊急バックアップ成功時のみ実行
- JSONバックアップは写真本体を含まず、軽量な本体データとして出力

## 注意

IndexedDBの写真は、SafariのWebサイトデータ削除で消える可能性があります。
写真本体の外部バックアップが必要な場合は、元写真をiPhone写真アプリ側にも残してください。

## 維持している固定ルール

- 保存キー：echeveria_pwa_data
- 直前データ：echeveria_pwa_previous_data
- 旧キー自動採用なし
- 保存データ診断・復旧候補欄なし
- localStorage.clear() 未使用
