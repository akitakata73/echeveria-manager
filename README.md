# エケベリア管理 PWA v38 Cultivar Filter

## v38の目的

株一覧ページに、状態フィルターと品種フィルターの2段構成を追加した版です。

## 主な変更点

- 株一覧のフィルターを2段化
  - 1段目：状態
  - 2段目以降：品種
- 品種フィルターは折り返し表示
- 品種フィルターは使用中の品種のみ表示
- 品種フィルターには件数を表示しない
- 品種フィルターに「すべて」を追加
- 状態フィルターと品種フィルターは排他制御
- 品種選択後の株一覧は、選んだ品種の中で次回確認日順に表示
- 品種チップは、ひらがな → カタカナ → 漢字 → その他の順を基本に並べる

## 維持している固定ルール

- 保存キー：echeveria_pwa_data
- 直前データ：echeveria_pwa_previous_data
- 旧キー自動採用なし
- 保存データ診断・復旧候補欄なし
- localStorage.clear() 未使用
