# エケベリア管理 PWA v36 UI Confirm

## v36の目的

v35 Data Guardを安定版として維持し、保存方式には触らず、実用UIだけを改善した版です。

## 主な変更点

- カレンダーの土日色分け
  - 土曜：青〜水色系
  - 日曜：赤系
  - 選択中の日付は選択表示を優先
- 保存・追加・記録後のトースト文言を「保存しました」に統一
- 主要保存ボタンの二重押し防止
  - 新規株登録／株情報保存
  - 水やり記録
  - 観察記録
  - 写真追加
  - マスター追加・編集

## 維持している固定ルール

- 保存キー：echeveria_pwa_data
- 直前データ：echeveria_pwa_previous_data
- 旧キー自動採用なし
- 保存データ診断・復旧候補欄なし
- localStorage.clear() 未使用
