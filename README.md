# エケベリア管理 PWA v43 Overdue Only Photo Cleanup

## v43の目的

v42の確認ラベルを整理し、写真なしプレースホルダーの扱いを改善した版です。

## 主な変更点

- 「本日確認」ラベルを削除
- 確認ラベルは「確認目安超過」のみ表示
- 遅れ日数は引き続き表示しない
- 1枚目の実写真を追加した際、同じ株の「写真なし」系プレースホルダー写真メタを自動削除
- 写真一覧では、実写真がある場合は「写真なし」系プレースホルダーを表示しない

## 維持している固定ルール

- 保存キー：echeveria_pwa_data
- 直前データ：echeveria_pwa_previous_data
- 写真本体：IndexedDB
- 旧キー自動採用なし
- 保存データ診断・復旧候補欄なし
- localStorage.clear() 未使用
- v40 Safe Commit の保存確定処理を維持
- v41 Photo Stability の写真保存安定化を維持
