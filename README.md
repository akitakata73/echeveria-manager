# エケベリア管理 PWA v13

## v13の主な変更点

- 右上ボタンを「＋株」から更新ボタンに変更
- ホーム画面に新規株登録ボタンを残し、右上との重複を解消
- 観察後の次回確認日がホーム画面へ反映されるよう、更新導線を追加
- 観察画面で状態変更時に次回確認日を再計算
- 観察画面・水やり画面・株情報編集画面の要観察フラグ連動を再点検
  - 良好：要観察OFF
  - 良好以外：要観察ON
- 入力途中の画面移動に確認メッセージを追加
  - 新規株／株情報編集：管理名、位置メモ、メモ
  - 水やり：メモ
  - 観察：観察メモ
- メモ等が未入力の場合は確認メッセージなし
- 戻る操作・タブ移動・更新操作でも入力保護を適用

## 注意

iPhone/Safari標準の確認ダイアログでは、ボタン文字色を赤・青に指定できません。
文言は「移動する場合はOK、入力を続ける場合はキャンセル」としています。
