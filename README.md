# エケベリア管理 PWA v1

## これは何か

益田議員のエケベリア管理用に作成した、無料運用前提のPWA実用版v1です。

App Store配布ではなく、Safariで開いて「ホーム画面に追加」して使う想定です。

## 主な機能

- 株登録
- 株一覧
- 株詳細
- 水やり記録
- 水やり量レベル
- 任意mL記録
- 水やり方法
- 次回確認日
- 観察記録
- 品種・鉢・用土・置き場所マスター
- 株一覧CSV
- 履歴CSV
- JSONバックアップ
- JSON復元

## 使い方

1. `index.html` をWeb上に置く
2. iPhoneのSafariでURLを開く
3. 共有ボタンから「ホーム画面に追加」
4. ホーム画面のアイコンから起動

## 無料で公開する候補

- GitHub Pages
- Cloudflare Pages
- Netlify
- Vercel

## 注意

データはブラウザの localStorage に保存されます。  
Safariのデータ削除や端末トラブルで消える可能性があります。

定期的に「設定」→「出力」→「JSONバックアップ」をコピーして保存してください。
