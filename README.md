# エケベリア管理 PWA v6

## v6の主な変更点

- 次回確認日の試験的な自動計算ロジックを追加
- 計算理由を画面上に表示
- 即日提案を避け、最低1日後に補正
- 小数点は切り捨て
- AM7時をホーム画面内通知の基準に設定
- 17時以降の水やりは +0.5日補正
- 季節補正を追加
  - 3月：±0
  - 4〜6月：-0.5
  - 7〜9月：-1.0
  - 10月：-0.5
  - 11月：±0
  - 12〜2月：+1.0
- 日当たり（方角）補正を追加
  - 南向き：-0.5
  - 北向き：+0.5
  - 東向き／西向き／室内照明：±0
- 状態補正を追加
  - 蒸れ注意：-0.5
  - 休眠気味：+0.5
- 「方角」を「日当たり（方角）」に名称変更
- 日当たり（方角）の選択肢に「室内照明」を追加

## 補正上限

- 成株：±2.0日
- 成株以外：±1.0日

## 注意

今回の自動計算は試験実装です。
水やり日ではなく「次回確認日」の提案です。実際に水やりするかはユーザー判断です。
