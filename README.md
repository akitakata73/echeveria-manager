# エケベリア管理 PWA v9

## v9の主な変更点

- 鉢マスターに「乾きやすさ」を追加
- 用土マスターに「乾きやすさ」を追加
- 鉢・用土の乾きやすさを次回確認日の補正に反映
- 補正は5段階、0.5日刻み
  - かなり乾きやすい：-1.0日
  - やや乾きやすい：-0.5日
  - 普通：±0日
  - やや乾きにくい：+0.5日
  - かなり乾きにくい：+1.0日
- 既存データは、鉢・用土とも乾きやすさ「普通」として自動移行

## 注意

補正項目が増えるため、最終補正上限は従来どおり維持しています。

- 成株：±2.0日
- 成株以外：±1.0日

そのため、鉢・用土・季節・日当たり・状態補正をすべて足しても、過度に短く・長くなりすぎない設計です。
