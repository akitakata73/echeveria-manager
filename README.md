# エケベリア管理 PWA v11

## v11の主な変更点

- 水やり量による次回確認日の補正を追加
- 水量補正を計算理由に表示
- 水やり画面で水量を変更すると次回確認日が再計算される
- 株情報編集画面の最終水やり量を変更しても次回確認日が再計算される

## 水量補正

- なし：-1.0日
- ごく少量：-1.0日
- 少なめ：-0.5日
- 普通：±0日
- 多め：+0.5日
- たっぷり：+1.0日

## 注意

補正項目がさらに増えるため、最終補正上限は従来どおり維持しています。

- 成株：±2.0日
- 成株以外：±1.0日

そのため、短期管理株で水量・季節・鉢・用土などの補正が重なっても、極端な日数になりにくい設計です。
