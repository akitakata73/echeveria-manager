# エケベリア管理 PWA v37 Clone From Add

## v37の目的

新規株追加ページから、既存株を元にしたクローン作成へ入れるようにした版です。

## 主な変更点

- 新規株追加ページの最上部に「既存株からのクローンを作成」ボタンを追加
- ボタン押下後、既存株一覧からクローン元を選択
- クローン元を選ぶと、コピー済みの新規登録フォームへ移動

## クローン時にコピーする項目

- 品種
- 鉢
- 用土
- 置き場所
- 位置メモ
- 個体区分
- 状態
- メモ

## クローン時にコピーしない項目

- 写真
- 最終水やり日
- 最終水やり量
- 次回確認日

## 維持している固定ルール

- 保存キー：echeveria_pwa_data
- 直前データ：echeveria_pwa_previous_data
- 旧キー自動採用なし
- 保存データ診断・復旧候補欄なし
- localStorage.clear() 未使用
