# エケベリア管理 PWA v26 Stable

## v26の主な変更点

- 保存キーを本番用の固定キー `echeveria_pwa_data` に変更
- 旧バージョン・復旧版の保存キー探索は維持
- ホーム画面のバックアップ状況カードを削除
- 記録ページのバックアップボタンは維持
- 設定画面の「出力・バックアップ」カードを削除
- 設定画面のバックアップ状況カードは維持
- 設定画面のバージョン表示・読込元・保存データ診断は維持

## 削除時の安全確認

以下の内部処理は削除していません。

- openExportCenter()
- openExportPreview()
- openImport()
- backupCardHtml()
- JSONバックアップ生成
- JSON復元

## 更新後の確認

1. `?v=26-stable` で開く
2. 設定画面で `現在のバージョン：v26-stable` を確認
3. 読込元キー、株数、履歴、写真が正常か確認
4. 記録ページのバックアップボタンが残っているか確認
5. 設定画面のバックアップ状況が残っているか確認
